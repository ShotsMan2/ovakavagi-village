# Konya Ovakavağı Village Website 🌾

This project is a modern, single-page website designed to introduce the beauties, culture, and lifestyle of Ovakavağı village, located in the Karatay district of Konya, Turkey.

## 🚀 Features

* **Responsive Design:** Flawless display on both mobile devices and desktop screens.
* **Modern CSS:** Contemporary design created using asymmetrical cuts (clip-path), CSS Grid, and Flexbox layouts.
* **Content Sections:** About, Agriculture & Life, Gallery, and Google Maps integration.
* **Pure HTML/CSS:** Coded entirely with Vanilla HTML and CSS, without the use of any external libraries or frameworks.

## 🛠️ Technologies Used

* HTML5
* CSS3
